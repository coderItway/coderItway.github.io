# 一键把掘金等文章生成 md 文件

## 推荐使用的工具 Clean−mark

这是一个简单实用的工具。针对你的需求，它可以将博客文章转换为干净的 Markdown 文本文件。

## 安装

`npm install clean-mark --global`

## 下载文章

`clean-mark "链接文章" -o /Users/aaa/Desktop/xxx.md`

`"链接文章" `：文章的 url

`/Users/aaa/Desktop/`: 下载的路径

`xxx.md`：文件名, 后缀.md 可加可不加, 会自动生成.md 文件
